# Assistance Weatherlane

Pour obtenir de l'aide, poser une question ou nous faire part de vos commentaires, écrivez-nous à **support@studentdriver.co**. Nous nous efforçons de répondre sous 1 à 2 jours ouvrés.

Lorsque vous nous contactez, il est utile d'inclure :

- Le modèle de votre iPhone et la version d'iOS
- La version de Weatherlane
- Une brève description de ce qui s'est passé

## Questions fréquentes

**Comment gérer mon abonnement Weatherlane Pro ?**
Ouvrez l'app Réglages, touchez votre nom, puis Abonnements, puis Weatherlane Pro. Vous pouvez y changer de formule ou résilier.

**Mon widget ne se met pas à jour.**
Ouvrez l'app une fois pour qu'elle puisse actualiser les prévisions en cache. Si l'affichage reste figé, touchez le widget de manière prolongée, choisissez Modifier le widget, et confirmez que le bon lieu est sélectionné.

**Pourquoi la température semble-t-elle incorrecte ?**
Ouvrez le menu de l'app et vérifiez si vous êtes en Celsius ou en Fahrenheit. Nous utilisons par défaut la convention de votre région, mais le réglage est là si vous souhaitez en changer.
